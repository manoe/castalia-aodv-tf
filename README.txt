Steps to install the AODV

1. Intall Omnet-4.x

2. Install Castalia-3.2

3. Copy the folder AODV to the Castalia folder. E.g. home/user/Castalia-3.2  

5. Open the prompt, go to the AODV folder

6. Type sh install.sh
	# Answer with "n" to any question that appear on the console

7. Go to the path Castalia-3.2/Simulations/AODVTest and type the following command: Castalia -c General

8.  Enjoy :)
